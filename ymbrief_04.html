🟢 1. 교감신경 활성약

🔸 직접 교감 활성 (수용체 직접 결합)

Epinephrine (에피네프린) α1·α2·β1·β2 모두
모든 아드레날린 수용체를 전부 자극 — 아나필락시스 응급 처치의 핵심
Phenylephrine (페닐에프린) α1 선택적
코막힘 제거제 (비충혈 제거) · 승압제
→ α1 수축 → 혈관수축으로 혈압↑, 코점막 혈관수축
Dobutamine (도부타민) β1 선택적
급성 심부전 시 심수축력 강화
→ β1 자극 → 심근 수축력 ↑
Albuterol / Terbutaline β2 선택적
천식 · COPD 치료 — 기관지 확장
→ β2 자극 → 기관지 평활근 이완
Isoproterenol (이소프로테레놀) β1·β2 (비선택)
심박출력 강화 (β1) + 혈관 확장 → 혈압 하강 (β2)
반사 빈맥 주의

🔸 혼합·간접작용 아드레날린 약물

Amphetamine / Methylphenidate / Tyramine
소포 내 NE를 밖으로 실어냄 (역배출)
→ NET(재흡수 수송체)의 기질 + 역수송으로 NE 방출 ↑
💊 임상: ADHD 치료 (Methylphenidate)
Cocaine / TCA (삼환계 항우울제)
NET 차단 → 시냅스 내 NE 증가
→ NE가 수용체에 더 오래 작용
💡 직접 vs 간접 구분 포인트
직접: 수용체에 스스로 결합 (Epi, Phenylephrine 등)
간접: NE 방출 ↑ 또는 재흡수 ↓ 로 내인성 NE를 증폭
혼합: 두 기전 모두 보유

🔴 2. 교감신경 억제약 (α·β 차단약)

🔸 α 차단약

Phenoxybenzamine α (비가역)
비가역적 α 결합 (공유결합)
💊 임상: 크롬친화세포종 (Pheochromocytoma) 고혈압 치료
기립성 저혈압 주의 · 반사 빈맥 주의
Prazosin / Tamsulosin α1 선택적
BPH (전립선 비대증) 배뇨장애 개선
고혈압 치료
→ α1 차단 → 요도·방광경부 평활근 이완, 혈관 이완
Clonidine / Methyldopa α2 효능 (중추성)
중추성 교감 억제로 고혈압 치료
→ 중추 α2 자극 → NE 방출 억제 → 말초 교감 ↓
💡 Methyldopa는 임신성 고혈압에 안전

🔸 β 차단약

Metoprolol / Atenolol / Esmolol β1 선택적 (심장 선택적)
심장 억제 → 고혈압 · 부정맥 치료
💡 Esmolol은 반감기 10분 초단기 → 응급용
기관지 수축 부작용 (β2 억제 잔여)
Propranolol β1·β2 (비선택)
심장 억제 (β1) + 기관지 수축 (β2)
천식환자 금기
Labetalol / Carvedilol α1 + β (혼합)
심장 억제 (β) + 혈관 확장 (α1 차단)
⭐ Carvedilol은 심부전 사망률 감소에 중요!
⚠️ β 차단약 공통 부작용·주의사항
• 서맥 유발
• 당뇨환자 저혈당 경고 증상 (빈맥·떨림) 은폐 → 위험
• 갑작스런 투약 중단 금지 → 반동성 빈맥 · 허혈 악화 (MI 유발 가능)
• 천식 · COPD 환자는 비선택성 β차단약 금기

🟠 3. 부교감 억제약 (항콜린제 / M 수용체 차단)

Atropine (아트로핀) M 비선택 (3급 아민)
💊 임상: 서맥 치료 · 산동제 (동공 확대) · 농약 해독
💡 3급 아민 → BBB 통과 (중추 작용 O)
요저류 악화 주의
Scopolamine (스코폴라민) M 비선택 (3급 아민)
💊 멀미 예방 패치
💡 3급 아민 → 중추 작용
Ipratropium / Tiotropium M (흡입)
💊 천식 · COPD — 기관지 확장
→ 기도 평활근 M3 차단
Oxybutynin (옥시부티닌) M3 (방광)
💊 과민성 방광 — 배뇨근 이완
⚠️ 항콜린제 공통 주의사항
• 녹내장 악화 (동공 산대 → 방수 배출 차단)
• 전립선 비대 환자 요저류 악화
• 장폐색 악화
🧠 아트로핀 중독 5대 증상 암기: "건·확·빠·홍·미"
건 구강건조 · 확 동공확대(산동) · 빠 빈맥 · 홍 홍조·체온상승 · 미 초조·섬망
💊 치료: Physostigmine (3급 AChE 억제제로 BBB 통과 가능 → 중추 역전)

🔵 4. 부교감 활성약 (직접 콜린성 — M 수용체 효능)

Bethanechol (베타네콜) M (장·방광)
💊 장폐색 · 요저류 치료
→ 장·방광 평활근 수축 → 운동성 촉진
Pilocarpine (필로카르핀) M (눈·침샘)
💊 녹내장 치료 (동공축소 → 방수 배출 ↑)
💊 구강건조 (Sjögren 증후군) — 침샘 자극

🟣 5. AChE 억제제 (간접 부교감 활성)

아세틸콜린 분해효소를 억제하여 시냅스 내 ACh를 증가시킴.

🔸 가역적 AChE 억제제

Neostigmine / Pyridostigmine 4급 아민 (BBB 통과 ✕)
💊 중증 근무력증 (Myasthenia Gravis) 치료
💡 4급 아민 → 말초에만 작용
→ Carbamylation (가역적 공유결합)
Donepezil / Rivastigmine 3급 아민 (BBB 통과 ✓)
💊 알츠하이머 인지장애 개선
💡 3급 아민 → 중추신경계 통과 → 뇌 ACh ↑
🧠 3급 vs 4급 아민 암기
3급 아민 = 지용성 → BBB 통과 ✓ → 중추·말초 모두
→ Atropine, Scopolamine, Physostigmine, Donepezil, Rivastigmine
4급 아민 = 전하 有 → BBB 통과 ✕ → 말초만
→ Neostigmine, Pyridostigmine

☠️ 6. 유기인계 농약 · 신경가스 (비가역적 AChE 억제)

유기인계 농약 · 신경가스 비가역적 AChE 억제
예: Echothiophate · Sarin
비가역적 인산화 → 심각한 독성 유발
💡 시간이 지나면 "Aging(노화)" 발생 → 영구적 비활성화
⚠️ 유기인 중독 증상 (콜린성 과활성): "SLUDGE + BB"
Salivation (침흘림) · Lacrimation (눈물) · Urination (요실금) · Defecation (설사)
GI distress (복통) · Emesis (구토)
Bronchospasm (기관지경련) · Bradycardia (서맥)
💊 유기인 중독 해독제 — "Atropine + PAM 투여"
1. Atropine (3급 아민)
  → 무스카린 증상(SLUDGE) 억제 — 경쟁적 M 차단
  → 4급은 중추 통과 못하므로 3급 필수

2. Pralidoxime (PAM / 2-PAM)
  → AChE 재활성화 (인산기 떼어냄)
  → 단, "노화(aging)" 전에만 효과!

3. Benzodiazepine — 경련 억제

📊 7. 수용체 요약표 & 암기법

🔸 수용체별 주요 작용

수용체G단백질주요 작용예시 약물
α1 Gq 혈관수축, 산동, 요도수축 Phenylephrine (+) / Prazosin (−)
α2 Gi 교감 억제 (중추), NE 방출 ↓ Clonidine (+)
β1 Gs 심수축력 ↑, 심박수 ↑ Dobutamine (+) / Metoprolol (−)
β2 Gs 기관지 이완, 혈관 확장 Albuterol (+)
M2 Gi 심박수 ↓ (심장 부교감) Atropine (−)
M3 Gq 기관지 수축, 분비 ↑, 동공축소 Pilocarpine (+) / Ipratropium (−)

🔸 한눈에 보는 약물 분류 지도

분류기전대표 약물
교감 직접 활성 수용체 직접 결합 Epinephrine, Phenylephrine, Dobutamine, Albuterol, Isoproterenol
교감 간접·혼합 NE 방출↑ / 재흡수↓ Amphetamine, Methylphenidate, Tyramine, Cocaine, TCA
교감 α 억제 α 수용체 차단 Phenoxybenzamine, Prazosin, Tamsulosin
교감 β 억제 β 수용체 차단 Metoprolol, Atenolol, Esmolol, Propranolol, Carvedilol, Labetalol
부교감 억제 (항콜린) M 수용체 차단 Atropine, Scopolamine, Ipratropium, Tiotropium, Oxybutynin
부교감 직접 활성 M 수용체 효능 Bethanechol, Pilocarpine
부교감 간접 활성 AChE 억제 (가역) Neostigmine, Pyridostigmine, Donepezil, Rivastigmine, Physostigmine
AChE 비가역 억제 유기인 — 독성 Echothiophate, Sarin, 농약
🎯 시험 빈출 포인트 총정리
① Esmolol = 반감기 10분 → 응급 사용
② Carvedilol = 심부전 사망률 감소
③ Phenoxybenzamine = 크롬친화세포종 (비가역 α 차단)
④ 3급 아민 (Atropine, Physostigmine, Donepezil) = BBB 통과
⑤ 4급 아민 (Neostigmine, Pyridostigmine) = 말초만
⑥ 유기인 해독 = Atropine + PAM (노화 전!)
⑦ β 차단약 = 급성 중단 금지 (반동성 허혈)
⑧ 아트로핀 중독 5대 증상 = 건·확·빠·홍·미
